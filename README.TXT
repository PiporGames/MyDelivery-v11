MyDelivery-v11 es un proyecto de Java que consiste en la emulación de la gestión de un restaurante en tiempo real.
El restaurante está compuesto de diferentes partes, como la cocina, los robots que dispensan los ingredientes, los motoristas que entregan los pedidos, los bancos para registrar las cuentas y finanzas...
Cada clase/entidad del proyecto realiza una tarea en el contexto del restaurante y, más importante, se comunican de diferentes maneras entre ellos.
El proyecto está creado con diferentes tecnologías de concurrencia, como Mutex, CyclicBarriers, ... y comunicación, como TCP, UDP y gRPC. 
El objetivo del proyecto es familiarizarse con los aspectos y comprenden el diseño correcto y eficiente de aplicaciones concurrentes y distribuidas.
Este proyecto ha sido creado para la asignatura de Programación Concurrente y Dirigida del Grado de Ingeniería en Ingeniería del Software de la UEx año 2023/2024.  

Creado por José Manuel de Torres Domínguez.  

En este repositorio se encuentra el proyecto de Eclipse for Java y el metadata del workspace.
Para su ejecución es recomendable utilizar las configuraciones de ejecución ya diseñadas.
Es necesario tener instalados todos las dependencias que pida el proyecto, evidentemente.
